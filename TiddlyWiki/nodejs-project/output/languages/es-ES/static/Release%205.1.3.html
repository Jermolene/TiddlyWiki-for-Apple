Release 5.1.3

 21 febrero 2015 a las 23:01

See GitHub for detailed change history of this release

Usability Improvements

  • Added "new journal" button; see Creating journal tiddlers
  • Added "new here" button; see Creating and editing tiddlers
  • Added "new journal here" button; see Creating journal tiddlers
  • Made date format strings be translateable
  • Added Russian translation
  • Added confirmation when clearing password
  • Added additional prompt when setting password
  • Increased size of the clear search button
  • Upgraded to KaTeX v0.1.1, with support for several additional LaTeX features

Hackability Improvements

  • Added audio "parser" for handling Audio content
  • Added support for ActionWidgets
  • Added new addprefix and addsuffix operators
  • Added support for custom password prompts
  • Extended the list-links macro to display caption field if present
  • Extended ImageWidget to allow percentage width and height to be specified

Bug Fixes

  • Fixed the each operator to work with missing tiddlers
  • Fixed problem with tiddler width in zoomin storyview with the sidebar hidden
  • Fixed bug whereby the tm-home message wasn't navigating to a tiddler, causing problems in zoomin storyview
  • Fixed problem with jumping toolbar icons under Firefox
  • Fixed problem with untagged label being incorrectly coloured
  • Fixed problem with title background colours with the "Sticky Titles" theme
  • Fixed problem with subfilter parameter of timeline macro
  • Exclude search string tiddler from search results

Node.js Changes

These changes are only relevant to people using TiddlyWiki under Node.js

  • Extended KaTeX plugin to be able to work under Node.js to generate static HTML
  • Extended "includeWikis" to merge build targets
  • Refactored the build scripts for tiddlywiki.com into a separate repository at https://github.com/Jermolene/build.jermolene.github.io
  • Extended boot kernel to raise an error if a missing wiki folder is specified on the command line
