WebServer Parameters

 2018년 6월 30일 오후 6:09

The following configuration parameters are supported by the integrated WebServer:

  • admin
  • anon-username
  • authenticated-user-header
  • credentials
  • csrf-disable
  • debug-level
  • host
  • password
  • path-prefix
  • port
  • readers
  • required-plugins
  • root-render-type
  • root-serve-type
  • root-tiddler
  • system-tiddler-render-template
  • system-tiddler-render-type
  • tiddler-render-template
  • tiddler-render-type
  • tls-cert
  • tls-key
  • tls-passphrase
  • username
  • writers
