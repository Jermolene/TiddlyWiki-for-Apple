WebServer API

Get the content of a static file. See using the integrated static file server

GET /files/<pathname>

Parameters:

  • pathname - URI encoded path to the file

Response:

  • 200 OK
    Content-Type: <content-type> (determined from file extension)
    Body: data retrieved from file
  • 403 Forbidden
  • 404 Not Found
